# 🌟 Data Analytics & Machine Learning Portfolio

Selamat datang di *repository* portofolio data saya. Di sini, saya mendokumentasikan berbagai studi kasus analisis data, mulai dari pemrosesan data mentah, visualisasi *dashboard* interaktif, hingga pemodelan prediktif (*Machine Learning*).

---

## 📁 Katalog Proyek

### 🐍 Python & Machine Learning (Jupyter Notebook / Colab)
Kumpulan proyek yang berfokus pada *Data Mining, Data Cleaning*, dan pembuatan algoritma cerdas menggunakan bahasa pemrograman Python.

* **[1. Retail Mall Customer Analytics](./Mall_Customer_Analytics)**
  * **Deskripsi:** Penerapan algoritma *K-Means Clustering* untuk segmentasi pelanggan dan *Linear Regression* untuk memprediksi potensi pengeluaran pengunjung mal.
  * **Format Utama:** *Source Code* (`.ipynb`)
  * **Tools:** Python, Pandas, Scikit-Learn, Matplotlib.

---

### 📊 Data Visualization & Business Intelligence (Tableau)
Kumpulan proyek yang berfokus pada *Data Storytelling* dan pembuatan metrik bisnis melalui *dashboard* interaktif.

* **[2. E-Commerce Digital Product Sales](./Ecommerce_Dashboard)**
  * **Deskripsi:** Analisis performa penjualan produk digital untuk menentukan "Star Product" dan mengukur efektivitas *channel marketing* (WhatsApp vs TikTok).
  * **Format Utama:** *Dashboard Image* (`.png`) & Laporan Bisnis
  * **Tools:** Tableau Public, Microsoft Excel.

* **[3. Social Media Content Performance](./Social_Media_Analytics)**
  * **Deskripsi:** Analisis *Engagement Rate* (ER) dan efektivitas durasi video lintas platform (TikTok, YouTube, Instagram) untuk merumuskan strategi konten berbasis data.
  * **Format Utama:** *Dashboard Image* (`.png`) & Laporan Bisnis
  * **Tools:** Tableau Public.
